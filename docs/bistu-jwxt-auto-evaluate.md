# BISTU教务系统一键评教脚本

## 脚本信息
- **名称**: BISTU教务系统一键评教
- **版本**: 3.0.0
- **作者**: MRZZJ
- **许可证**: MIT

## 功能介绍
- 🚀 一键自动选中所有评教选项
- 🔍 智能检测评教元素，无元素时给出清晰提示
- 🎨 醒目按钮样式，固定在页面右侧，不遮挡操作
- 🔄 适配单页应用路由变化，页面跳转后自动重建按钮

## 安装方法
1. 浏览器安装油猴插件（Tampermonkey/Greasemonkey）
2. 点击 [一键评教脚本](https://raw.githubusercontent.com/CN-MRZZJ/tampermonkey-scripts/main/bistu-jwxt-auto-evaluate.user.js) 安装

## 使用说明
1. 登录教务系统，进入具体的评教条目页面
2. 点击右侧的「一键评教」按钮
3. 等待提示“评教完成”后，手动点击页面提交按钮即可
4. 若按钮未显示，可刷新页面重试

## 注意事项
> ⚠️ 由于教务系统可能进行了更新，此功能可能暂时不可用
- 脚本仅负责辅助操作，最终提交需手动确认
- 若按钮未显示或功能失效，可能是因为教务系统页面结构发生了变化
- 如遇到问题，可查看浏览器控制台的错误信息

## 技术实现
- 使用MutationObserver监控页面变化
- 智能检测评教表单元素
- 自动填充评教选项
- 适配单页应用的路由变化

## 常见问题
1. **按钮不显示**
   - 可能是因为页面结构发生了变化
   - 尝试刷新页面或重新安装脚本

2. **评教未完成**
   - 脚本只会自动填充选项，不会自动提交
   - 请手动点击页面上的提交按钮

3. **功能失效**
   - 可能是因为教务系统进行了更新
   - 请检查脚本是否需要更新
